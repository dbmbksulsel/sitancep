# SIBILING 

#Deskripsi
Aplikasi SIBILING merupakan kependekan dari Aplikasi Sistem Informasi Bimbingan dan Konseling. Sesuai dengan namanya aplikasi ini digunakan untuk mencatat data pelanggaran siswa di sekolah dengan mudah, cepat dan sederhana. Tidak lagi diperlukan buku pelanggaran karena datanya telah tercatat rapi melalui aplikasi ini. Fasilitas yang ditawarkan dalam aplikasi ini masih sangat sederhana.

#Dibuat dengan :
1. jQuery
2. Twitter Bootstrap versi 3
3. jQuery datepicker (untuk mengambil data tanggal)

#Versi 1.3 (Juli 2017)
1. Input data siswa, kelas, jurusan, pelanggaran, sanksi, ortu, guru, pelanggaran siswa
2. Cetak surat data pelanggaran siswa
3. Rubah password user
4. Memindahkan kelas berkelompok
5. Terdiri dari 4 level user (admin, siswa, ortu, guru)
